# Release 1.1.0

Дата: 2026-02-19

## Ключевые улучшения
1. **Операционная готовность**
   - Добавлен шаблон конфигурации `.env.example`.
   - Документация приведена к реальной структуре репозитория.

2. **Производительность базы данных**
   - Включены `foreign_keys` и режим `WAL`.
   - Добавлены индексы для часто используемых запросов:
     - активные товары;
     - заказы пользователя;
     - пополнения пользователя;
     - поиск по invoice id;
     - username пользователей.

3. **Качество и поддерживаемость**
   - Внедрены базовые unit-тесты для критичных утилит (парсинг сумм, экранирование, парсинг админ-ID).
   - Добавлены dev-зависимости для воспроизводимого запуска тестов.

## Команды проверки
```bash
pip install -r requirements-dev.txt
pytest
```

## Upgrade notes
- Обновление обратносуместимо с текущей SQLite-схемой.
- Индексы создаются автоматически при `db.init()`.


## Patch 1.1.1
- Починена модульная структура импортов для текущего layout репозитория (убраны некорректные relative-импорты уровня `..`).
- Добавлен smoke-тест импортов для `main`/`handlers`/`keyboards`/`utils`.
