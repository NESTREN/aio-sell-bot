# Changelog

## [1.1.1] - 2026-02-19
### Fixed
- Исправлены импорты во всех ключевых модулях: проект теперь корректно запускается из текущей структуры репозитория (`python -m main`) без package-relative конфликтов.

- Добавлены release-документы `CHANGELOG.md` и `RELEASE.md`.

- SQLite-инициализация усилена: включены `PRAGMA foreign_keys`, `WAL` и индексы для ускорения каталога/покупок/поиска.
